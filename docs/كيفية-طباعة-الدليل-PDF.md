# كيفية طباعة دليل المستخدم كـ PDF

الدليل جاهز بصيغتين — اختر الطريقة الأنسب لك.

---

## الطريقة 1 — من ملف HTML (الأفضل للطباعة الاحترافية)

1. افتح الملف في **Chrome** أو **Edge**:
   ```
   docs/دليل-المستخدم.html
   ```
   (انقر مرتين على الملف أو اسحبه للمتصفح)

2. من القائمة: **طباعة** (Ctrl+P)

3. الوجهة: **حفظ كـ PDF** أو **Microsoft Print to PDF**

4. الإعدادات المُوصى بها:
   - **الحجم:** A4
   - **الاتجاه:** عمودي
   - **الهوامش:** افتراضي
   - ✅ **رسومات الخلفية** (Background graphics) — مفعّلة

5. احفظ باسم مثل: `دليل-ماستر-كلينك-بلس.pdf`

> النتيجة: غلاف احترافي + فهرس + فصول منسّقة جاهزة للتسليم للعميل.

---

## الطريقة 2 — من ملف Markdown

1. افتح `docs/دليل-المستخدم-ماستر-كلينك-بلس.md` في:
   - **VS Code / Cursor** مع معاينة Markdown، أو
   - **Typora**، أو
   - موقع [markdowntopdf.com](https://www.markdowntopdf.com)

2. صدّر أو اطبع كـ PDF

---

## الطريقة 3 — رفع الدليل على الإنترنت (اختياري)

يمكنك رفع ملف PDF على:
- Google Drive / OneDrive — مشاركة رابط مع العملاء
- موقع عيادتك — صفحة «دليل الاستخدام»
- إرساله مع عقد البيع كمرفق

---

## للمطوّر — تضمين رابط في العقد

عند تسليم النظام للعميل، أرفق:

| الملف | الاستخدام |
|-------|-----------|
| `docs/دليل-المستخدم.html` | طباعة PDF للعميل |
| `docs/دليل-المستخدم-ماستر-كلينك-بلس.md` | تعديل المحتوى لاحقاً |
| رابط `/login` | نقطة البدء للعميل |

---

## تحديث الدليل

عند إضافة ميزات جديدة:
1. عدّل ملف `.md`
2. حدّث ملف `.html` بنفس المحتوى (أو اطلب من المطوّر مزامنتهما)
